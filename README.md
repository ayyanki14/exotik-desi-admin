# Exotik-Desi — Admin Dashboard
Phase 2: Private admin panel with products, orders, notify list, AI insights.

## Deploy to Vercel
1. Push this folder to a GitHub repo  
2. Import repo in Vercel dashboard
3. Framework: Vite
4. Build command: npm run build
5. Output directory: dist

## URLs
- Production: exotikdesi-admin.vercel.app
- Keep this URL private — only share with store owner
